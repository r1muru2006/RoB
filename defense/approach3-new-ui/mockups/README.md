# New UI Design Mockups

Comparison of the current (old) FSA API permission dialog boxes vs. the proposed new designs from Section 6.3 of the paper.

## Key Changes

### Read Permission Dialog (Figure 4)
- Added warning icon
- Added "and its subdirectories" text
- Added "might attempt to steal your sensitive information" warning
- Added "Get more information" and "Report suspicious" links

### Write Permission Dialog (Figure 5)
- Added warning icon
- Added "and its subdirectories" text
- Added "permanent loss of your local data" warning
- Added expandable list of impacted files
- Added "Get more information" and "Report suspicious" links

## Viewing
Open `generate_mockups.html` in any browser to see the side-by-side comparison.
